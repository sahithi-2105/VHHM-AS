# VHHM-AS: Digital Twin Driven Virtual Human Health Monitoring and Alert System

A state-of-the-art preventive healthcare platform that creates a virtual health profile (Digital Twin) to monitor and alert users of potential health risks.

## 🚀 Project Overview

The **Virtual Human Health Monitoring System (VHHM-AS)** uses AI and the Digital Twin concept to help users track and improve their health effortlessly. By collecting basic health metrics, the system generates a dynamic virtual profile that reflects the user's current condition and provides early warnings, healthcare reports, and lifestyle suggestions.

## 🏗️ Technical Architecture

### Frontend (`/vhhm-frontend`)
- **Next.js 14/15:** High-performance React framework.
- **Three.js / React Three Fiber:** Specialized 3D visualization for the Virtual Human Digital Twin.
- **Framer Motion:** Smooth micro-animations for a premium feel.
- **Recharts:** Data-driven health monitoring trends.

### Backend (`/vhhm-backend`)
- **Python / FastAPI:** High-speed API for AI processing.
- **Scikit-learn / SciPy:** Machine learning for risk detection and health assessment.
- **LangChain / NLP:** Conversational health companion chatbot.
- **PostgreSQL:** Reliable storage for user health history.

## ❤️ Key Features
- **AI Health Chatbot:** Conversational data collection and guidance.
- **Dynamic Digital Twin:** A visual representation of health status.
- **Early Alert System:** Real-time push notifications for critical health parameters (BP, SpO2, etc.).
- **Lifestyle Recommendations:** Data-driven suggestions for water intake, sleep, and activity.

## 🛠️ Getting Started

### Backend
1. `cd vhhm-backend`
2. `pip install -r requirements.txt`
3. `python -m app.main`

### Frontend
1. `cd vhhm-frontend`
2. `npm install`
3. `npm run dev`
